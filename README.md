# -Logistic-regression-predicts-Titanic-survival

一个基于 Titanic 数据集的机器学习入门项目。  
本项目使用 **Jupyter Notebook** 完成数据清洗、探索性分析、特征工程，并基于 **逻辑回归模型** 预测泰坦尼克号乘客的生存情况。

---

## 项目简介

泰坦尼克号沉船事件是数据分析与机器学习中的经典案例之一。  
本项目根据乘客的个人信息和乘船信息，例如：

- 性别
- 年龄
- 船舱等级
- 家庭成员数量

构建逻辑回归模型，分析哪些因素会影响乘客的幸存概率，并进一步对测试集中的乘客进行生存预测。

---

## 分析目标

本项目主要完成以下任务：

1. 对 Titanic 训练数据进行读取与清洗
2. 处理缺失值、数据类型和无效字段
3. 通过可视化探索各变量与幸存率之间的关系
4. 构建逻辑回归模型分析影响因素
5. 使用训练好的模型对测试集乘客进行生存预测

---

## 数据集说明

项目包含两个数据文件：

- `titanic_train.csv`：训练数据，包含乘客是否幸存的标签
- `titanic_test.csv`：测试数据，不包含幸存标签，用于预测

主要字段包括：

- `PassengerId`：乘客ID
- `Survived`：是否幸存
- `Pclass`：船舱等级
- `Sex`：性别
- `Age`：年龄
- `SibSp`：同乘伴侣 / 同胞数量
- `Parch`：同乘父母 / 孩子数量
- `Fare`：票价
- `Cabin`：船舱号
- `Embarked`：登船港口

---

## 项目流程

### 1. 数据读取
使用 Pandas 读取训练集数据，对整体结构和字段信息进行查看。

### 2. 数据清洗
对数据进行预处理，包括：

- 填充 `Age` 缺失值
- 转换分类变量的数据类型
- 检查重复值与异常值
- 保留不参与建模但可用于理解数据的字段

### 3. 特征工程
构造新的特征：

- `FamilyNum = SibSp + Parch`

用于表示乘客同乘家庭成员数量，以进一步分析家庭规模与幸存率之间的关系。

### 4. 探索性数据分析
通过图表分析以下变量与幸存率之间的关系：

- 幸存人数分布
- 年龄分布
- 票价分布
- 船舱等级与幸存关系
- 性别与幸存关系
- 登船港口与幸存关系
- 家庭成员数量与幸存关系

### 5. 逻辑回归建模
使用逻辑回归模型对乘客幸存情况进行分析。  
在建模前，对分类变量进行哑变量处理，并筛除部分不纳入模型的字段。

最终模型重点考察的变量包括：

- 年龄
- 家庭成员数量
- 船舱等级
- 性别

### 6. 测试集预测
对 `titanic_test.csv` 进行与训练集一致的预处理后，使用训练好的逻辑回归模型预测乘客幸存概率，并将概率大于等于 0.5 的样本判定为幸存。

---

## 分析结论

根据模型结果，可以得到以下结论：

- 年龄越小的乘客，幸存概率相对更高
- 女性乘客的幸存概率明显高于男性乘客
- 船舱等级越高，幸存概率越高
- 同乘家庭成员数量较少的乘客，幸存概率相对更高

这些结论与泰坦尼克号事件中“妇女儿童优先”和不同舱位逃生条件差异的历史背景基本一致。
